# KaSupply

KaSupply is a marketplace app that connects buyers with trusted local suppliers.

## Local development

1. Install dependencies.
2. Create `kasupply-app/.env.local` with your Supabase credentials.
3. Start the app:

```bash
npm run dev
```

The app runs at [http://localhost:3000](http://localhost:3000) and now redirects the root route to the buyer homepage at `/buyer`.
